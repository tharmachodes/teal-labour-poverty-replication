# Teal Paper Replication: Labour Prices and Poverty

This repository contains materials for replicating and extending the analysis from the Teal paper (2000) on the relationship between labour prices and poverty traps in developing countries.

## Repository Structure

- `data/household_survey.csv`: Anonymized household survey data used in the analysis.
- `original_analysis.R`: R script reproducing the main regression table and Figure 2.
- `docs/Teal_etal_2000.pdf`: The original paper (provided for reference).
- `replication_log.md`: Log of replication attempts.
- `extension_analysis.R`: Script for proposed extension analysis.

## Purpose

This repository is intended for educational use as part of a course module on the micro-foundations of poverty. Students are expected to replicate the core findings and propose a theoretically-motivated extension.
